Haunted Webring

a haunted webring. this cgi program implements a double-linked list that you can query. i'm hosting this in tokyo, and you're free to use it! or, if you prefer, you can build and host it yourself! the source code is available on [github]

here is the [full list] of domains in the webring right now

usage

usage: before | after | random | list

?before

GET webring.cgi?before=https://pixeldreams.tokyo/

redirects to the site immediately BEFORE https://pixeldreams.tokyo/

?after

GET webring.cgi?after=https://pixeldreams.tokyo/

redirects to the site immediately AFTER https://pixeldreams.tokyo/

?random

GET webring.cgi?random=https://pixeldreams.tokyo/

redirects to a random site in the ring EXCEPT https://pixeldreams.tokyo/

?list

GET webring.cgi?list

returns the hard-coded list of domains in the webring in text/plain

html tag

paste this into your html and modify how you like! (don't forget to edit the domain part)

  <a href='https://pixeldreams.tokyo/cgi-bin/webring.cgi?before=<your domain>'>&lt;&lt;&lt;</a>
